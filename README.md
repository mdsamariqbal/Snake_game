Snake Game made with Python and Pygame.
Eat apples, grow longer, and avoid hitting yourself.

How to run:
	1.	Install pygame
pip install pygame
	2.	Run the game
python snakegame.py

Controls:
	•	Arrow keys to move
	•	Y to play again
	•	N to quit

Features:
	•	Snake grows when you eat
	•	Score increases
	•	Self-collision ends the game

Have fun!
